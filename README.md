# 🚀 Run and Deploy

This contains everything you need to run your app locally.

---

## 🛠️ Run Locally

### 📌 Prerequisites:
- Node.js installed

### ⚡ Steps:

1. Install dependencies
```bash
npm install
```
2. Set your API key in .env.local
```bash
GEMINI_API_KEY=your_api_key_here
```

3. Run the app
```bash
npm run dev
```
